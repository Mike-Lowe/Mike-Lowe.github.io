What is Javascript?

Imagine a cake. At first it's a bowl of ingredients. We could say that this is the html.

Then there are the decorations: The icing, the figures on top, the candles. These are representations of CSS. The style that adds colour and character to the cake.

But then there is the magical, unseen element that baked the cake and brought it to life. This is Javascript. Javascript enlivens html elements like how heat acts as a catalyst in the baking process. No heat - no cake. No Javascript - no fully functioning website.

What is Control Flow?

Your computer runs code from top to bottom and Control Flow is the expression of that behaviour. What changes the direction of the control flow are things such as Loops, Conditionals, or Functions.

As can be seen from the diagram above. Control flow follows a fundamentally simple pattern of logic.

To use a culinary example again - it is like making a meal. There is a method to the preparation, but there are various recipes and ingredients within those recipes that can be used before the final product is served.

Loops

A loop is a tool that repeats a set of instructions. It employs the term "iterate", which bascially means "repeat" and will occur until an outcome (referred to as a stopping condition) is met.

The above diagram represents what happens in a Javascript loop.

Below is a an illustration of a simple loop that outputs the numbers two through to five. The "i" is a variable index followed by a condition that checks if "i" is less than or equal to five, followed by the increment "++", which adds one to the operand "i".

What does this mean? As in the first diagram, the equation will "loop" until the specified conditions are met.

Arrays and Objects

Objects represent a special data type that is mutable and can be used to store a collection of data (rather than just a single value).

The example above illustrates how dot notation accesses "key: value" attributes in an object.

Arrays are used to create lists of items in a single variable. They are useful when creating ordered clusters of data that can then be referred to by their numerical position in the list.

Arrays employ zero-based indexing, so the first item in an array has an index of 0, the second item an index of 1, and so on.

In the example above, the item 'apple' is indexed at position 2 in the array.
