What is the difference between Margin, Border and Padding?

The above diagram displays the demarcations that illustrate the CSS box model. It is a method by which a box is wrapped around an HTML element.

Description of various parts:

  • Content - Where the text and visual content of the box is displayed
  • Padding - Makes a transparent space around the content
  • Border - Borders the padding and content
  • Margin - Similar to padding in that it makes transparent space around the border
